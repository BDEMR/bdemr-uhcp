# bdemr-doctor-app
BDEMR Doctor App


## Installing

```sh
npm install
```

```sh
[sudo] npm install gulp -g
```
```
sudo npm install mimosa -g`
```

## Running

```sh
gulp
```
or `mimosa watch --server`

## Building

```sh
gulp
```
or `mimosa build -mo`

## Notes
app build files/folders should be coppied to 'app' folder while creating new production commit.
